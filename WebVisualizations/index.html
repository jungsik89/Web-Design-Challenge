Summary: Latitude vs X


The goal of this data analysis was to see how the weather changes as you get closer to the equator. To select a large sample size of the cities near the equator, first we assembled a dataset of 500 cities, we then extracted the data from OpenWeatherMap API. After we assembled the dataset, we integrated Matplotlib to plot the various elements of weather vs latitude of those 500 cities. Four different elemental setting was chosen for this analysis: Humidity,Max Temp, Cloudiness, Wind Speed. This website provides the source data used for this analysis and I dive in visually and explain the analysis how the weather dramatically changes near the equator.
